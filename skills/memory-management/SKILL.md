# 记忆管理技能

## 使用场景
当用户分享了重要信息、个人偏好、重要日期或其他值得长期保存的内容时使用此技能。系统会自动判断信息价值并智能分类存储。

## 核心功能

### 1. 智能保存
- 自动判断信息重要性，过滤无效内容
- 区分短期记忆和长期记忆
- 支持用户明确指定长期记忆

### 2. 智能检索
- 根据当前对话上下文检索相关记忆
- 使用语义相似度匹配，而非简单关键词
- 优先显示最相关的记忆

### 3. 记忆压缩
- **日常记忆**：30天后自动压缩成主题聚类，使用标准相似度阈值(0.7)
- **长期记忆**：90天后进行温和压缩，使用更宽松阈值(0.6)，保留更多原始信息
- 生成记忆摘要和关键洞察，减少存储空间同时保持信息价值

### 4. 洞察分析
- 分析用户的记忆模式和行为偏好
- 识别经常提及的主题和关注点
- 提供个性化的记忆洞察

## 使用步骤

### 保存记忆
1. 分析对话内容，识别有价值信息
2. 使用 `saveMemory` 工具自动保存
3. 对明确的重要信息使用 `saveLongTermMemory`

### 检索记忆
1. 当需要参考历史信息时，使用 `retrieveRelevantMemories`
2. 传入当前对话上下文作为检索条件
3. 系统返回最相关的记忆列表

## 注意事项
- 系统会自动过滤无效和琐碎内容
- 记忆会智能聚类，避免重复存储
- 旧记忆会自动压缩，保持系统性能
- 检索结果按相关性排序，优先显示最匹配的内容

## 示例工具使用

```typescript

// 检索相关记忆

  context: "用户界面偏好",
  limit: 5
});

// 获取记忆洞察
```
